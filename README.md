# RGB Color Game

A game to guess color from its RGB values.

## Tech Used

* HTML
* CSS
* JAVASCRIPT


## Features
- Different Levels for varying difficulty (easy, medium, hard)
- Option to restart game again even if not completed
- Concise Display
